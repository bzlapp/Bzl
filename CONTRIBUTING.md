# Contributing

Thanks for helping improve Bzl.

## Where to file issues

Please follow: `docs/ISSUE_TRACKER.md`

## Development setup

1. Install Node.js (recommended: Node 18+)
2. Install deps:
   - `npm.cmd install` (Windows PowerShell)
   - or `npm install` (macOS/Linux, or Windows with `npm` available)
3. Run:
   - `npm.cmd start` (Windows PowerShell)
   - or `npm start`

## Pull requests

- Keep PRs focused (one fix/feature per PR when possible).
- Include a clear description and reproduction steps (or screenshots/video for UI changes).
- If you touched the UI, include before/after notes and any relevant browser console output.
- If you touched plugins, update `docs/PLUGINS.md` when the API changes.

## Security issues

Please do not file public issues for vulnerabilities. See `SECURITY.md`.
